03 — Design Direction

Visual Identity & Aesthetic

The design follows the fluid.glass aesthetic — a premium, editorial approach with dark/cream palette, cinematic imagery, generous whitespace, and sophisticated motion design.

Design Reference

Primary inspiration: fluid.glass — an Awwwards-recognized architectural glazing website. Key patterns adopted: bottom-fixed glassmorphic navigation, 3D cube preloader, full-viewport sections, word-by-word text reveals, clip-path image animations, and editorial grid layouts.

07 — Development Timeline

Major Prompts & Iterations

Phase 1 — Initial Analysis
Document Absorption & Website Analysis
Read RFQ PDF (22 modules), SOW text file. Fetched and analyzed eastandwest.ae — extracted all content, images, colors, tone, navigation structure, property listings, contact info.
Phase 2 — Sitemap Creation
Homepage Section Architecture
Created detailed sitemap: Preloader → Announcement Bar → Header → Hero → About → Properties → Managed Properties → Services → Values → Locations → Blog → Newsletter → Footer.
Phase 3 — First Build (Rejected)
Initial Homepage Build
Built with traditional top header, Playfair Display font, standard card layouts, basic GSAP animations. Client feedback: "the website looks very bad", "current website looks better than this one".
Phase 4 — Complete Rebuild
fluid.glass Aesthetic Adoption
Analyzed fluid.glass in extreme detail. Rebuilt everything from scratch: bottom-fixed glassmorphic nav, 3D cube preloader, dark/cream palette, editorial layouts, Syne + Space Mono fonts.
Phase 5 — Image Integration
Local Images + Unsplash
Copied 4 client images from /images folder to public. Added Unsplash images for hero (Dubai skyline), showcase (glass architecture), gallery, and footer (luxury property).
Phase 6 — Animation Overhaul
Premium Animations + New Sections
Added 6 new sections (Marquee, Intro, Vision, Stats, FeaturedProjects). Upgraded all animations: word-by-word reveals, clip-path image reveals, parallax, GSAP timeline transitions. Added Lenis smooth scroll + custom cursor (later removed for performance).
Phase 7 — Performance Optimization
Scroll Lag Resolution
Removed Lenis (double RAF loop), removed custom cursor (backdrop-filter GPU overhead), removed 8 scrub-based ScrollTriggers (Gallery per-image parallax, Stats parallax, Reviews gradient). Scroll became smooth.
Phase 8 — Visual Polish
Accent Color, Graphics, Font, Header, Footer
Integrated #cee002 across all hover states, buttons, links, gradients. Added decorative graphic elements (rotating squares, dot grids, gradient orbs, corner brackets). Switched to Outfit font. Redesigned hero with top header bar + logo. Rebuilt footer with fluid.glass dark card menu layout. Fixed nav overlap with IntersectionObserver.
Phase 9 — Deployment
Code Deployment
Committed all work and pushed to the private remote repository on main branch.
08 — Key Design Decisions

Decisions & Rationale

DecisionChoiceRationale
Navigation styleBottom-fixed floating barMatches fluid.glass reference, modern and distinctive
Color themeDark/cream with #cee002 accentCream backgrounds from fluid.glass, accent inherited from client brand
FontOutfit (→ Aeonik Pro alternative)Closest free geometric sans-serif to Aeonik Pro; premium feel
Animation libraryGSAP + ScrollTriggerIndustry standard for premium web animations, fine-grained control
Smooth scrollNative (Lenis removed)Lenis caused double RAF + conflicts with 15+ ScrollTrigger instances
Custom cursorRemovedbackdrop-filter on moving element caused frame drops
Image strategyLocal + Unsplash4 client photos for properties, Unsplash for hero/showcase atmosphere
Footer designFull-viewport with dark cardMatches fluid.glass footer — background image, centered nav card
Responsive approachFlexbox + CSS Grid inlineAvoided Tailwind/CSS grid conflicts from v1; reliable across viewports

Issues Encountered & Resolved

IssueResolution
White screen on load (v1)Removed position:fixed body + scroll container pattern; moved fonts to link tags
Grid-24 class conflicts with Tailwind col-span-*Replaced with inline flexbox/grid styles using px values
Scroll lag with Lenis + 11 scrub triggersRemoved Lenis, reduced scrub triggers from 11 to 4
Bottom nav overlapping footerIntersectionObserver hides nav when footer is 15% visible
GSAP tl.call() type error (4 args)Changed to tl.add() which accepts position parameter
Aeonik Pro unavailable (commercial font)Substituted with Outfit from Google Fonts
External images slow to loadDownloaded EWIG logo + property images locally to /public
